## 简介

## 模块

| 模块                |     介绍                                                                          |
| -------------------|---------------------------------------------------------------------------------- |
core|核心，常用的工具类
domain|定义通用的实体类
exception|定义通用的异常类
extra|扩展模块，对第三方库的封装，比如 Spring、Mybatis Plus 等等
web|web模块
